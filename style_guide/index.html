Markup

I used Sass to keep my code as clean and organized as possible. Below are some examples of code used on the site.

Grid Mixin

Buzz is built on a custom, 12 column grid base.


$tot-width: 100%;
$tot-num-col: 12;
$gut-width: 10%;
$col-width: ($tot-width - (($tot-num-col - 1)*$gut-width))/$tot-num-col;

@mixin grid($col-num) {
	float: left;
	width: ($col-width*$col-num)+(($col-num - 1)*$gut-width);
	margin-right: $gut-width;

	&:last-child {
		margin-right: 0;
	}
}

Rem Conversion Mixin

I used a mixin to convert to pixels and provide a backup for browsers that don't support rem.


@mixin fntz($rem){
	size: (($rem*16) + px);
	size: ($rem + rem);
}

Color Variables

I used variables to define color so I can easily make changes at any time.


$color-text: #2e2c2c;
$color-primary: #f6ad22;
$color-secondary: #c37012;
$color-tertiary: #6b804f;
$color-dark: #544834;
$color-light: #f5f3ec;

Font Variables

I imported Google fonts into a Sass partial and used variables to define type styles.


@import url(https://fonts.googleapis.com/css?family=Alegreya:900,700|Lato:400,700);

$font-headings: "Alegreya", serif;
$font-main: "Lato", sans-serif;
